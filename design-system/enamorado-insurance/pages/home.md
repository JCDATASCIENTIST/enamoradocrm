# Page Override: Home (website `/en`, `/es`)

**Pattern:** Hero + Trust + Social proof + FAQ + CTA (adapted from Hero + Testimonials)

## Section order

1. Trust bar (phone, service area)
2. Hero — broker intro, **amber primary CTA** (Schedule consultation), secondary outline
3. Agency statement (single column, high contrast)
4. How we help — 4 cards with numbered badges
5. Process — 4 steps, amber step numbers
6. FAQ — accordion-style list + schema
7. CTA band — phone + contact

## Rules

- Hero CTA: `accent-600`, min 44px height
- Phone always visible: trust bar + mobile sticky call bar
- No video hero (accessibility; static gradient + pattern OK)
- Card hover: shadow only, no transform scale
