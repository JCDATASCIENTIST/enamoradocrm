# Admin user guide — Enamorado CRM

## Daily workflow

1. **Dashboard** — Review overdue renewals, pending follow-ups, and commission totals.
2. **Clients & prospects** — Full list with filters; export CSV or open print view.
3. **Pipeline** — Drag cards between stages or use arrow buttons on each card.
4. **Follow-ups** — Complete, skip, or snooze tasks from the daily buckets.

## User management

1. Go to **Users** (admin only).
2. **Invite user:** email, name, role, temporary password (min 8 characters).
3. Change role or deactivate users from the table (you cannot deactivate yourself).

## Audit log

**Audit Log** shows insert/update/delete on contacts, profiles, follow-ups, commissions, and enrollments. Filter by table, action, or date range. Entries are append-only.

## Notes and data integrity

- Agents can add notes but cannot edit or delete them.
- Admins can edit or delete notes from the contact detail page.
- Admins can delete a contact from the detail page (permanent).

## Workstream B

- **Renewals** — Tabs for overdue, due within 7 days, and upcoming within 30 days.
- **Commissions** — Create and track by contact; filter by status.
- **Enrollments** — Log enrollment activity with status and external reference.

## HIPAA Path B

Do not enter SSNs or full Medicare/Member IDs. Only last-4 member ID is allowed.
