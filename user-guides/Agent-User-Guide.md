# Agent user guide — Enamorado CRM

## Your permissions

- View all contacts and notes.
- Create contacts and add notes.
- Edit contacts **assigned to you** or **unassigned**.
- Move pipeline cards for contacts you are allowed to edit.
- Log commissions and enrollments for contacts you can edit.

You cannot delete contacts, edit others' notes, or access **Users** or **Audit Log**.

## Recommended daily flow

1. Open **Follow-ups** — work overdue and today buckets first.
2. Open **Pipeline** — advance prospects through New → Requested → In Progress → Done.
3. Open a contact — add a note after every call or email.
4. Use **Birthdays** for outreach planning.

## Creating a contact

**Clients & Prospects → + New contact.** Required: first and last name. Set type (prospect/client), stage, and assignee.

## Promoting a prospect

On the contact detail page, use **Promote to client** when they become a client.

## Printing and export

- **Print view** from the contacts list filters.
- **Export CSV** uses the same filters (assignee shown by name).
