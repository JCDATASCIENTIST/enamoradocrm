# Agent Monday morning checklist

One-page routine for insurance agents using Enamorado CRM.

## Before clients (15 minutes)

- [ ] Log in at `crm.enamoradoinsurancecompany.com`
- [ ] Open **Follow-ups** — work **Overdue** first, then **Today**
- [ ] Open **Pipeline** — move cards that advanced since Friday
- [ ] Check **Renewals → Due (7 days)** for any policy renewals this week

## During the day

- [ ] After every call or email: add a **Note** on the contact
- [ ] Update **follow_up_date** when scheduling next touch
- [ ] New lead → **+ New contact** as **prospect**, assign yourself, set follow-up
- [ ] Won business → **Promote to client** → start **Enrollment** record

## End of day (5 minutes)

- [ ] Clear or snooze remaining follow-ups you touched
- [ ] Move pipeline stages to reflect reality (don't leave stale **In Progress**)
- [ ] Log commissions when carrier confirms ( **Commissions** page)

## Rules to remember

- No full Medicare/Medicaid IDs — last 4 digits only on member ID field
- You can edit contacts assigned to you or unassigned; not other agents' assigned contacts
- You cannot delete contacts or access **Users** / **Audit Log**
